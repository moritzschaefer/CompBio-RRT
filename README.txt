Roblib Planning Tutorial

Install:
- Copy this Folder and its contents into %ROBLIB_ROOT%\examples
- Add the follwing line to %ROBLIB_ROOT%\CMakeLists.txt:
	ADD_SUBDIRECTORY(examples/tutorialPlan)
	
Build:
- Compile the roblib library, the tutorialPlan project should be compiled with it

Run:
- Go to %ROBLIB_ROOT%\%BUILD_DIR%\examples\tutorialPlan and execute the binary

Further Information:
- Further Information about the Tutorial can be found on the robotics lab wiki: 	
	http://elektro.robotics.tu-berlin.de/index.php/Roblib_plan_tutorial
